# Password-Generator

Problem
Who has this problem?

People trying to think of a secure password.



Value
What benefit will your project provide?

Provide a simple web based password generator that doesn't need an app installed.



Smallest Demonstration of Value
What is the smallest version of your application that proves your idea works?

A website that allows the user to select the password length and allowed characters with a simple UI.


Required Features
What features are necessary to deliver that value?

A secure pseudo-random character generator.


Stretch Features
If you finish early, what would you add?

A way to encrypt/decrypt messages.
